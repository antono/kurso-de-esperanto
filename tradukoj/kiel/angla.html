When you launch the course, you will see the following items in the first window:


What is Esperanto?  Description of Esperanto.


How to use this course:  This file.


Settings:  Select this option to configure the Esperanto Course:


       Translation:  This Esperanto Course can be taken in several languages.


               Language:  Select the language to be used in the

               course from those in the list;


               Font:  Font to be used for text in the national

               language. A font that includes the special Esperanto

               characters is preferable.  The default font is Verdana Eo.

               Note:  Some fonts may exceed the maximum size of

               the fields.


Special characters in the e-mailed exercises:  If you have selected a tutor, this section allows you to select which system will be used to represent the special Esperanto characters in the exercises which are sent by e-mail to your tutor.


Unicode: In this system, your exercises will be sent with the special Esperanto characters. (for example. Ĉ and ŭ).  This is the default system for the course and is recommended.


System "x" (ŝ = sx): In this system the special letters are designated by adding an “x” after the letter.  So ŝ becomes sx.  This is an alternative method used when the student's e-mail program doesn't support Esperanto’s special characters.


System "h" (ŝ = sh): Similar to the “x” system except that we use an “h” instead of the “x”.

Course Tutor:  Enter the data related to your tutor.  If you still don't have a tutor, go to http://www.kurso.com.br/korektanto.php?en 


Diacritical marks:  Select between using the system of the course (valid only inside the program "Kurso de Esperanto") or another system already installed in the computer.  For example, to type any letter that has a “hat” above it (such as ĉ or ŭ), type  “^”, then type the letter.

Default:  the system included in the program.




Credits: Type of user license for the program, name of the author and collaborators, methodology, information on copyrights for the material used in the course, name of the translator (if available)


Exit: To leave the program.



Lessons Menu:


Lessons 01 to 07: Lessons with pronunciation and auditory understanding exercises.


Lessons 08 the 12:  The rest of the lessons, without pronunciation, but with varied exercises.


Fina Ekzameno:  Final examination for the course.


Where did I stop:  Page marker.  Will go back to the last page used.



Recommendations to the student:


1.  Study according to your rhythm and available time. Don't go too fast.  Don't go beyond one lesson a day.  Go ahead only when you know the content of the lesson well.


2.  Repeat the exercises as many times as you feel necessary. Constant practice leads to perfection.


3.  Read the Esperanto words and sentences aloud, repeating when necessary, even to the point of learning them by heart.


4.  The student may improve his learning by finding a tutor to correct the translation exercises, and to clarify any issues. To get a tutor, select the “Settings" option on the Main menu.


5.  After finishing the course, try to improve your knowledge by reading books and magazines in Esperanto, attending meetings of your local club if one exists, participating in discussions, internet chat, Esperanto Congresses, and engaging in correspondence with foreign countries.  This provides a fun way to improve your Esperanto, and at the same time you can make new friendships and extend your knowledge of other cultures.


6.  Join your local Esperanto organization, or your national organization.

You will enjoy many advantages such as discounts on books and magazines, and you will be

helping to spread the Esperanto language.

7.  Buy the basic reference books:  An Esperanto-English/English-Esperanto dictionary and an Esperanto grammar.  Consult the book service of your national organization to see what is available.


8.  Finally, take advantage of this admirable instrument of communication that is the Esperanto language, and become a citizen of the world!
